## Meta field

valid field everywhere
__typename: name of the object type of parent field in which is specified
    e.g. can use to handle data from field of union type, e.g. search results

```graphql
union SearchResult = A | B # ... other types
```

```graphql
{
  search(text: "foo") {
    __typename
    ... on A {
        # fields if A
    }
    ... on B {
        # fields if B
    }
    # other types
  }
}
```
